# ほんだな 文言ガイドライン v1（Step 3.7-1）

作成日: 2026-03-08
対象: 親向けUI / 子ども向けUI

## 目的

- 同じ意味の言葉を画面ごとに揺らさず、学習コストを下げる。
- 子どもモードは読字負荷を下げるため、表記ルール（ひらがな優先）を統一する。

## 基本方針

### 親向け

- 一般的な日本語UI語彙を使う（例: 読書中、読了、コメント、メモ）。
- 管理操作（更新・削除）は動詞を明確にする。

### 子ども向け

- 原則ひらがな中心（小学低学年でも読める表現）。ただし、一般的なカタカナ語（ログイン、メッセージ等）は無理にひらがな化しない。
- 固有名詞・記号は例外許容（例: ISBN、PIN）。
- 難語を使う場合は言い換え優先（例: 著者 → かいたひと）。

## 用語辞書（正準語）

### ステータス

| 意味           | 親向け   | 子ども向け   |
| -------------- | -------- | ------------ |
| これから読む   | 読みたい | よみたい     |
| 途中まで読んだ | 読書中   | よんでる     |
| 読み終えた     | 読了     | よみおわった |

### 記録入力・詳細

| 意味         | 親向け            | 子ども向け       |
| ------------ | ----------------- | ---------------- |
| 記録         | 読書記録 / 記録   | どくしょきろく   |
| 感想自由入力 | メモ              | ひとことかんそう |
| 著者         | 著者              | かいたひと       |
| 読んだ日     | 読了日 / 読んだ日 | よんだひ         |

### ログイン

| 意味         | 親向け     | 子ども向け |
| ------------ | ---------- | ---------- |
| 子ども識別子 | 子どもID   | こどもID   |
| 認証コード   | パスワード | 4けたPIN   |

## 運用ルール

1. 新規UI追加時は本ガイドの正準語を使用する。
2. 文言変更PRでは、同義語の重複がないか `rg` で確認する。
3. 子ども向け画面で漢字を使う場合は「低学年で読めるか」をレビュー観点に含める。
